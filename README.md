# Anchor
Anchor is a modular rehabilitation device designed to support people with Fisher’s Syndrome during and after their recovery. It is a soft, 3D-printed squeezable tool with interchangeable support attachments—a hand strap and a finger ring—that can be used based on the user’s comfort.

The product is intended for everyday use, to help improve hand strength and mobility through simple squeezing exercises. It is lightweight, easy to carry, wear, and remove, making it suitable for the users. The design is also simple to manufacture, requiring minimal material and labour, making it appropriate for large-scale production.

The current prototype demonstrates the product’s main functionality and shows how the modular attachment system can adapt to different users while supporting hand rehabilitation.
